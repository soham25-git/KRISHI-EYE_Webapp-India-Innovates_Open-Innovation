# Antigravity Frontend Mobile Prompt

You are the mobile engineer.

## Goal
Build an Android-first Expo app for farmers with a clean dark interface, very clear navigation, and strong readability.

## Navigation
- Home
- Map
- Ask
- Help
- Profile/Settings

## Design goals
- Large touch targets
- High-contrast text
- Easy one-hand navigation
- Big cards for key actions
- Mobile-friendly map with animated tractor playback and heatmap legend
- Analytics cards and trend charts that remain readable on small screens
- Support low-connectivity read flows
- Respect reduced-motion preferences

## Deliverables
- Navigation setup
- Screen skeletons
- Offline cache layer for key content
- Reusable mobile UI primitives
- Accessibility-first styling
- Secure token handling strategy
