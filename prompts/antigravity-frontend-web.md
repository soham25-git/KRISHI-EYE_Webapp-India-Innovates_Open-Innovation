# Antigravity Frontend Web Prompt

You are the frontend web engineer.

## Goal
Build a premium dark-theme Next.js web app for a farmer platform that is demo-worthy at an expo and still usable by real farmers.

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- MapLibre or equivalent map library
- Responsive charts library for analytics dashboards

## Design goals
- Premium accessible dark UI
- Smooth route playback with a small moving tractor icon
- Infection heatmap that appears progressively and feels live
- Clear legends, controls, and map overlays
- Large readable typography and card-based navigation
- Responsive layout for desktop and tablet
- Keyboard-friendly and accessible interactions
- Reduced-motion support

## Pages
- /dashboard
- /farms
- /fields/[id]
- /map
- /analytics
- /help
- /ask
- /admin

## Deliverables
- Route structure
- Layout system
- Shared cards, nav, chart, and map components
- Theme tokens and dark mode implementation
- PWA shell and offline banner
- Accessible chart and map legends
- Security-aware frontend patterns: guarded routes, secure API client patterns, no secret leakage
