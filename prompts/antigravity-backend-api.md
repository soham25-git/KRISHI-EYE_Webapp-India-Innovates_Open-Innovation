# Antigravity Backend API Prompt

You are the backend API engineer.

## Goal
Build the product APIs for a farmer web/mobile platform with demo-ready map playback, analytics, support directory, AI integration, and strong access control.

## Requirements
- NestJS
- TypeScript
- Swagger/OpenAPI
- ORM with PostgreSQL
- DTO validation and tests
- Rate limiting and structured logging
- Role and ownership checks

## Core modules
- auth
- users
- farms
- fields
- tractors
- jobs
- telemetry
- analytics
- support
- admin
- ai-proxy
- audit

## Deliverables
- Controllers and services for docs/api-routes.md
- Demo seed endpoints
- Analytics summary endpoints
- Support directory filters
- Stable contracts for frontend map and chart UIs
- Security middleware and authorization guards
- Audit log hooks for sensitive changes
