# Antigravity AI Systems Prompt

You are the AI systems engineer.

## Goal
Build a grounded advisory service for Indian agriculture.

## Requirements
- FastAPI
- RAG-based architecture
- Source retrieval with citations
- Confidence scoring and abstention
- Multilingual-ready request structure
- Farmer-friendly output language
- Guardrails for risky agricultural advice
- Logging for evaluation and moderation

## UX expectations
- Keep answers simple first, expandable second
- Return source metadata and confidence with every answer
- Suggest next actions like Help, Call, or Save
- Integrate cleanly with dark-theme web and mobile chat screens

## Deliverables
- Service scaffold
- Retrieval pipeline shape
- Answer contract
- Ingest skeleton
- Evaluation checklist
- Moderation and escalation hooks
